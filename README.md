```
    ├── ... any photo files
```

## How to Add Photos to the Gallery

1. Copy your photos into the `pictures/` folder
2. Open `index.html` in a browser
3. The gallery **automatically detects** all photos — no code changes needed!

### Supported filenames (auto-detected):
- `IMG_0276.JPG` through `IMG_0400.JPG`
- `photo_1.jpg` through `photo_60.jpg`
- `image_1.jpg` through `image_60.jpg`

To support other names, add them to `PROBE_NAMES` array in `script.js`.
